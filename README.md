#  HR Analytics Dashboard

## Project Overview
This project transforms messy HR data into an interactive Excel dashboard with key metrics for HR decision-making.

## Key Features
- Data cleaning & standardization
- 4 interactive charts
- 3 slicers for dynamic filtering
- KPI cards showing key metrics

## Technologies Used
- Microsoft Excel
- PivotTables
- Slicers
- Advanced formulas (VLOOKUP, DATEDIF, TRIM,IFS etc.)

## Dashboard Components
- **KPI Cards:** Total Employees (150), Avg Salary ($72,457), Attrition Rate (22.7%), Avg Performance (64.4)
- **Charts:** Headcount by Dept, Avg Salary by Dept, Attrition Rate by Dept, Bonus by Performance Band
- **Slicers:** Department, Employment Status, Performance Band

## Key Insights
- IT has the most employees (36), Marketing has the highest avg salary ($81,008)
- Finance has the highest attrition rate (23.5%)
- Active employees have higher performance scores than those who left
